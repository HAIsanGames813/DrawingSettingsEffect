# 描画設定エフェクト
基本的な描画設定ができる「描画設定」エフェクトを追加するプラグイン

## 導入方法
ダウンロードしたファイルをダブルクリックして、表示される画面に従ってインストールしてください。
### [最新バージョンはこちらから](https://github.com/HAIsanGames813/DrawingSettingsEffect/releases/latest)

## 概要
基本の描画設定をまとめた映像エフェクトを追加するプラグイン  

## 使用方法
映像エフェクトの描画タブに「描画設定」が追加されます。  
以下の設定を変更できます  
・X座標  
・Y座標  
・Z座標  
・左右反転  
・上下反転  
・不透明度  
・拡大率  
・横方向の拡大率  
・縦方向の拡大率  
・ドット絵  
・X軸回転  
・Y軸回転  
・Z軸回転  
※exo出力には対応していません
